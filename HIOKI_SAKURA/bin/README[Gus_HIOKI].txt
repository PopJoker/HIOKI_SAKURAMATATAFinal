HIOKI_SAKURA 使用說明

1. 安裝必要環境

	在使用主程式前，請先安裝驅動與必要函式庫。
	
	步驟： 1. 點開 “driverSetup.msi” 2. 依照畫面完成安裝

	此步驟會安裝： - 通訊驅動 - 系統必要函式庫 - HIOKI 設備相關相依套件

2. 執行主程式

	完成驅動安裝後，請直接執行：

		HIOKI_SAKURA.exe

	程式會自動載入預設設定。

3. 功能簡介

	-   多通道測量
	-   即時資料顯示
	-   工程模式（Config 設定）
	-   自動記錄與錯誤提示
	-   多 Slot / 多 Channel 自動掃描

4. 檔案說明

	HIOKI_SAKURA.exe 主程式
	driverSetup.msi 驅動安裝包
	ChannelSettings.csv 工程模式設定檔
	Config 資料夾 系統設定

5. 使用流程（快速指南）

	1.  執行 driverSetup.msi

	2.  連接 HIOKI 量測儀器

	3.  後點開 HIOKI_SAKURA.exe

	4.  等待程式偵測設備

	5.  開始測量

	6.  注意事項

-   若無法偵測設備，請確認 COM Port 是否正確
-   若出現 Port Busy，請重新插拔 USB 或重啟儀器
